# 🐶 Doggy Discovery

A fun and interactive Flutter application that fetches random dog images from the **Dog CEO API**. Complete with a quirky **Random Activity Generator**, this app doesn't just show you a cute dog—it tells you exactly what they are up to! 🐾

---

## 🚀 Key Features

- **Real-time Image Fetching** 📸: Seamlessly fetches new dog images using `async/await` for smooth asynchronous operations.
- **Random Activity Generator** 🎲: Every time a new image is fetched, the app pairs it with a fun, randomized status or activity update.
- **Clean & Modern UI** ✨: Designed with beautiful modern aesthetics using `BoxShadow`, `BorderRadius`, and a striking `DeepOrange` call-to-action button.
- **Robust Error Handling** 🛡️: Built-in `try-catch` blocks ensure graceful error handling for internet connectivity issues.

---

## 📦 Packages

This project utilizes the following carefully selected packages from pub.dev to enhance functionality and design:

- **[`http`](https://pub.dev/packages/http)** 🌐: Essential for making robust HTTP GET requests to the Dog CEO API to fetch the dog images.
- **[`google_fonts`](https://pub.dev/packages/google_fonts)** 🔤: Implements modern typography (specifically the **Poppins** font family) to give the application a refined, polished, and professional look.

## 🛠️ How to Run

Follow these simple steps to run the application on your local machine:

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   ```
2. **Navigate to the project directory:**
   ```bash
   cd fun_app
   ```
3. **Get Flutter dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   Ensure you have an emulator running or a device connected, then execute:
   ```bash
   flutter run
   ```
